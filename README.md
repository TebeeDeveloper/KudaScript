# KudaScript
Cáo ống tre phiên bản Việt

## Giới thiệu chung
- Được sinh ra để giải quyết những rắc rối mà ngôn ngữ lập trình khác của người Việt mang lại
- Phù hợp cho người mới học, lâu năm, hoặc muốn tìm kiếm ngôn ngữ kịch bản mới mẻ

### Ví dụ mã nguồn
```lua
-- in lời chào
print "Cáo ống tre phiên bản Việt"
```
